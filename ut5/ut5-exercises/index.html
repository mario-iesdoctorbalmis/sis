¿Qué es curl -s?

curl es una herramienta de línea de comandos que realiza peticiones HTTP (entre otros protocolos) y muestra la respuesta en la terminal. La opción -s (silent) suprime los mensajes de progreso y errores de transferencia, mostrando únicamente el resultado. En este caso hace una petición GET a la API de ipinfo.io, que devuelve en formato JSON información sobre la IP pública desde la que se realiza la consulta.
